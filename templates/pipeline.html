Check DNA quality with FastQC

Run FastQC on two FastQ files to evaluate the DNA quality. FastQC generates an HTML file showcasing the DNA quality.

Improve DNA Quality with Trimmomatic

If DNA quality is poor, use Trimmomatic to remove adapters and enhace DNA quality.Run FastQC again to confirm improvements.

Genome Mapping with BWA

Use BWA to create a genome map (index) with “hg19.fa”. Utilize BWA mem to map FastQ files against the reference genome, generating a .sam file with mapping information.

Data Sorting and Cleaning with Picard

Employ Picard's functions (SortSam, FixMateInformation, AddOrReplaceReadGroups) for data organization and error reduction. Use
MarkDuplicates to remove duplicates, PrintReads to filter low-quality reads, ensuring data integrity.

Genetic Variant Calling with GATK

Use GATK's HaplotypeCaller tool to identify genomic variations. IndexFeatureFile and VariantFiltration refine variations, and the results are stored in VCF files.

ANNOVAR for Genomic Analysis

Convert filtered VCF files to AVinput format using annovar_convert. Utilize ANNOVAR_table to annotate genomic variations with the humandb database with output as csv.
