1. Check DNA or RNA Quality with FastQC

Run FastQC on two FastQ files to evaluate the DNA or RNA quality. FastQC generates an HTML file showcasing the DNA quality.

2. Improve DNA Quality with Trimmomatic

If DNA quality is poor, use Trimmomatic to remove adapters and enhance DNA quality. Run FastQC again to confirm improvements.

3. Genome Mapping with BWA

Use BWA to create a genome map (index) with "hg19.fa." Utilize BWA mem to map FastQ files against the reference genome, generating a .sam file with mapping information.

4. Data Sorting and Cleaning with Picard

Employ Picard's functions (SortSam, FixMateInformation, AddOrReplaceReadGroups) for data organization and error reduction. Use MarkDuplicates to remove duplicates, PrintReads to filter low-quality reads, ensuring data integrity.

5. Genetic Variant Calling with GATK

Use GATK's HaplotypeCaller tool to identify genomic variations. IndexFeatureFile and VariantFiltration refine variations, and the results are stored in VCF files.

6. ANNOVAR for Genomic Analysis

Convert filtered VCF files to AVinput format using annovar_convert. Utilize ANNOVAR to annotate genomic variations with the humandb database.

7. Final Steps

Run IndexFeatureFile again to optimize access to PASS-filtered VCF files. VariantFiltration further refines SNP variations. The annotated SNP and Indels variations that passed the filter are ready for analysis.
