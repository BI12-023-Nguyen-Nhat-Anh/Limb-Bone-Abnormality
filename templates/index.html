Input

File FastQ

Real-Life Applications:

    Early Detection

  • The software streamlines the process of identifying limb bone abnormalities at an early stage with precision, allowing healthcare professionals to promptly implement appropriate interventions and personalized treatment plans.

    • Advancements in Research

  • Scientists and researchers can leverage the software to conduct thorough investigations into the genetic foundations of limb bone abnormalities. This contributes to the wider scientific knowledge and understanding of these conditions.

Bioinformatic tools

5 bioinformatic tools are used for the analysis of limb bone abnormality patient sequence data

Trimomatic 0.39

Trim and crop Illumina (FASTQ) data as well as to remove adapters

Genome Analysis Toolkit

Include tools for variant calling, recalibration, downstream processing

Annovar

Utilize update-to-date information to functionally annotate genetic variants detected from diverse genomes

VCFtools

Manipulate and filter Variant Call Format (VCF) files

Picard

Mark duplicate reads, sort, index BAM(Binary Alignment Map) files

FastQC

Check the quality of DNA
