We optimize the data processing workflow.

A website supports processing sequence data of Limb Bone Abnormality patients

The methods employed in this website are based on the research paper “From FastQ data to high confidence variant calls: the Genome Analysis Toolkit best practices pipeline” by Geraldine A. Van der Auwera, Mauricio O. Carneiro, Chris Hartl, Ryan Poplin, Guillermo del Angel, Ami Levy-Moonshine, Tadeusz Jordan, Khalid Shakir, David Roazen, Joel Thibault, Eric Banks, Kiran V. Garimella, David Altshuler, Stacey Gabriel, and Mark A. DePristo.

Bioinformatic tools

5 bioinformatic tools are used for the analysis of limb bone abnormality patient sequence data

Trimomatic 0.39

trim and crop Illumina (FASTQ) data as well as to remove adapters

Genome Analysis Toolkit

include tools for variant calling, recalibration, downstream processing

ANNOVAR

utilize update-to-date information to functionally annotate genetic variants detected from diverse genomes

VCFTOOLS

manipulate and filter Variant Call Format (VCF) files

Picard

mark duplicate reads, sort, index BAM(Binary Alignment Map) files

FastQC

check the quality of DNA
